Що таке JavaScript?

JavaScript була створена для того, щоб "оживати сторінки".

Програми на цій мові називаються скриптами. Їх можна писати прямо на HTML-сторінці і вони автоматично компілюються при завантаженні сторінки.

Для запуску JS не потрібні спеціальні компілятори чи спеціальна підготовка. У цьому плані JS дуже відрізняється від інших мов програмування - Java.

Чому цю мову називають JavaScript?

Коли створили JS, тоді вона мала назву LiveScript. В той час була досить популярною мова Java, тому задля популярності було названо JavaScript, типу "молодша систра" Java

Сьогодні JavaScript виконується не тільки у браузері, але і на серверах, або на будь-якому пристрої де є - рушій JavaScript.

Браузер має вбудований рушій, який деколи називають "віртуальною машиною JavaScript".

  • V8 - Chrome та Opera
  • SpiderMonkey

Ці віртуальні машини потрібно запамʼятати

Як працюють рушії?
Рушій є складним, але принцип роботи простий.
  • Рушій (вбудований, якщо це браузер) читає ("розбирає") скрипт.
  • Потім він перетворую скрипт в байт код.
  • І потім байт код виконується, і дуже швидко.
Рушій застосовує оптимізації на кожеому етапі процесу. Він навіть слідкує за скомпільваним скриптом під час його виконання, аналізує дані, які проходять через скрипт, і оптимізує байт код на основі цих знань.
Що може вбудований в браузер JS

Сучасний JavaScript - це безпечна мова програмування. Вона не дає низькорівневого доступу до памʼяті чи процесора.

Можливості JS залежать від оточення де її застосують. Наприклад, Node.js підтримує функції, які дозвояють читати/записувати різні файли, здійснювати запити, та інше.

Вбудований в браузер JS може робити все що повʼязано з маніпуляцією веб-сторінки

Наприклад, JS може:

  • Додавати новий HTML-код на сторінку, змінбвати існуючий вміст та змінювати стилі.
  • Реагувати на дії користувача.
  • Відправляти запити.
  • Отримати і надсилати куки.
